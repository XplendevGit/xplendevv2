# Xplendev.com

**Xplendev.com** es una plataforma especializada en el desarrollo de sitios web y soluciones digitales, brindando una experiencia optimizada, escalable y de alto rendimiento. Nuestra misión es ofrecer herramientas tecnológicas que potencien la presencia digital de nuestros clientes.

## 🚀 Tecnologías Utilizadas

- **Next.js** → Framework de React con renderizado híbrido (SSR/SSG).
- **Tailwind CSS** → Framework de estilos para un diseño responsivo y eficiente.
- **Framer Motion** → Animaciones fluidas y optimizadas para mejorar la experiencia del usuario.
- **NodeMailer** → Servicio para el envío de correos electrónicos.
- **BotPress** → Plataforma de automatización e inteligencia artificial para chatbots.

## 📌 Características Principales

✅ **Sitios web optimizados y escalables**.
✅ **Diseño moderno y minimalista con Tailwind CSS**.
✅ **Implementación de animaciones fluidas con Framer Motion**.
✅ **Integración de formularios de contacto con envío de correos a través de NodeMailer**.
✅ **Chatbot AI mediante BotPress**.
✅ **Despliegue en Vercel para alto rendimiento y confiabilidad**.

## 📋 Requisitos Previos

Antes de iniciar, asegúrate de tener instalado lo siguiente en tu sistema:

- **Node.js** (versión 14 o superior)
- **npm**, **yarn** o **pnpm** como gestor de paquetes
- **Acceso a un servidor de correo SMTP** para NodeMailer

## 📥 Instalación

1. **Clonar el repositorio**

   ```bash
   git clone https://github.com/tu_usuario/xplendev.com.git
   cd xplendev.com
   ```

2. **Instalar dependencias**

   Con npm:
   ```bash
   npm install
   ```
   Con yarn:
   ```bash
   yarn install
   ```

3. **Configurar variables de entorno**

   Crea un archivo `.env.local` en la raíz del proyecto con las siguientes variables:

   ```env
   NEXT_PUBLIC_SITE_URL=http://localhost:3000
   NODEMAILER_HOST=smtp.tudominio.com
   NODEMAILER_PORT=587
   NODEMAILER_USER=tu_usuario
   NODEMAILER_PASS=tu_contraseña
   BOTPRESS_API_KEY=tu_clave_de_api
   ```

4. **Ejecutar el servidor en desarrollo**

   ```bash
   npm run dev
   ```
   o
   ```bash
   yarn dev
   ```

   Luego, abre en tu navegador `http://localhost:3000` para ver el sitio en acción.


## 🖥️ Despliegue en Producción

Este proyecto está diseñado para desplegarse fácilmente en **Vercel**.

1. **Crear una cuenta en Vercel** y conectar el repositorio.
2. **Configurar las variables de entorno en el panel de Vercel**.
3. **Desplegar con un solo clic**.
